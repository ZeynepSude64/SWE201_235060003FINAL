# SWE201_235060003FINAL
## GymManagementApp

Bu proje SWE203 Object Oriented Programming dersi kapsamında geliştirilmiştir.

### Class Tasarımı
- **Member**: Spor salonu üyelerini temsil eder.
- **Trainer**: Eğitmen bilgilerini tutar.
- **Membership**: Üyelik türü ve ücret bilgisini içerir.

Class’lar Single Responsibility Principle’a uygun şekilde tasarlanmıştır.
Her class ayrı bir dosyada yazılarak kodun okunabilirliği ve sürdürülebilirliği artırılmıştır.
